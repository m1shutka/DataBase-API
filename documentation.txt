1) Добаление товара на склад:
	1. Получаем информацию о товаре из скана;
	2. Используем метод find_add_cell() для определения ячейки для добавления;
	3. Сканим ячейку верифицируя ее;
	4. Ипользуем метод add() для добавления, true говорит об успехе, false в свою очередь о фиаско.

2) Взятие товара со склада:
	1. По идее с изначально должна быть инфа о товаре, которая дюпнулась с сервера;
	2. Используем метод find_get_cell() для определения ячейки для взятия;
	3. Сканим ячейку верифицируя ее;
	4. Сканим товар верифицируя его;
	5. Используем метод get() для извлечения товара со склада, true говорит об успехе, false в свою очередь о фиаско.

3) Компрессия товара по складу:
	1. Используем метод get_movements() для получения списка перемещений в формате: {type: [[cell_out, cell_in]...]...};
	Далее для каждого подсписка в списке перемещений по типам товаров:
		2. Сканим cell_out верифицируя ее;
		3. Используем метод get_items_in_cell() либо get_items_data() (по фильтру с идентификатором cell_out) для получения списка товаров в ячейке cell_out;
		4. Сканим товары для верификации;
		5. Скиним ячейку cell_in для верификации;
		6. Для каждого товара в списке полученном в п.3 вызваем метод move() осуществляя перемещение.
